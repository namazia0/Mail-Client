# Mail Client Project

A mail client with a graphical user interface that enables you to log in, read, write, delete, save, and send Gmail messages.

This project was developed as part of the Object-oriented Software Development lab at the University of Bonn by a team of four students (Felix Hartkopf, Johannes Daniels, Lukas Hamm, and I). 
Over the course of four weeks in Summer 2021, we built the application using Java, Gradle, JSON, XML, and TCP/IP.
